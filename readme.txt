Manual de usuario(profe) de la App de Notas:

1)listar tareas: 
ingresar por consola => node app.js listar.
Se listaran todas las tareas

2)Crear tarea y agregarla a la base de datos:
ingresar por consola => node app.js "titulo" "estado".
En lugar de titulo y estado puede ingresar lo que desee.

3)filtrar tareas por estado:
